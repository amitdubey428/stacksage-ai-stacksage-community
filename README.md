# StackSage Community

Welcome to the **StackSage Community** repository — the central place for bugs, feature requests, and questions related to [StackSage](https://stacksageai.com/).

> **Note:** This repository does not contain source code. It exists solely as a public issue tracker and community hub.

---

## What is StackSage?

StackSage is an open-source AWS cost & configuration audit tool that helps engineering teams detect wasteful infrastructure, security misconfigurations, and cost anomalies — without sending your data anywhere.

- **CLI-first.** Runs in your terminal or CI pipelines.
- **100% local.** No data leaves your environment.
- **Rich HTML reports.** Shareable, actionable findings.

## Before Opening an Issue

1. **Search first** — your issue may already be reported or answered.
2. **Be specific** — include your StackSage version (`stacksage --version`), OS, and Python version.
3. **One issue per report** — keeps things focused and actionable.
